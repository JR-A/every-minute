# every-minute
자바 안드로이드 웹&amp;앱 개발자(2020) - 파이널 프로젝트
SpringMVC기반 시간표 관리 커뮤니티 웹 어플리케이션
(에브리타임 클론코딩)

## 개발 기간
  - 2020.11.12 ~ 2020.12.12
 
## 개발 환경
  -  Windows 10 64bit
  -  JDK 1.8
  -  eGov Spring Framework 4.3.22 Release
  - Oracle 11g
  -  Apache Tomcat v9.0
  - Spring MVC
## 기술 스택
  - Ajax
  - 지도API
